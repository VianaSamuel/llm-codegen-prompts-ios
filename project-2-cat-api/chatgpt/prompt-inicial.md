## Prompt Utilizado

Crie um aplicativo iOS completo em Swift que consuma dados da API pública The Cat API (https://thecatapi.com/), exibindo imagens de gatos e informações relacionadas a cada imagem, produzindo código funcional, legível e organizado, seguindo boas práticas de programação em Swift, utilizando arquitetura modular e facilmente adaptável, separando claramente modelos de dados, view models, serviços de rede e views, implementando requisições assíncronas à API com tratamento de erros, cache básico de imagens para melhor desempenho, atualização dinâmica da interface ao receber os dados, interface intuitiva construída em SwiftUI ou UIKit, comentários claros explicando cada parte principal do código, e garantindo que o aplicativo funcione corretamente ao ser compilado, sem dependência de bibliotecas externas, mas estruturado para futura expansão de funcionalidades.
